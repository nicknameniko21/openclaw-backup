## Next Steps

1. **Product Hunt:** Reach out to 5-10 hunters 30 days before launch
2. **LinkedIn:** Start with 10 connection requests/day, personalize each
3. **Reddit:** Begin commenting (add value) before posting
4. **Email:** Verify email lists, set up domain authentication
5. **Referral:** Soft launch with power users, gather feedback

## Compliance Notes

- **CAN-SPAM:** Include physical address and unsubscribe in all emails
- **GDPR:** Only email business addresses, provide opt-out
- **Product Hunt:** Never ask for upvotes directly
- **LinkedIn:** Respect connection limits (100/week)
- **Reddit:** Follow subreddit rules, disclose affiliation

# Quick Reference Cards

## Cialdini's 7 Principles - Quick Application Guide

| Principle | Trigger | SaaS Application | Copy Example |
|-----------|---------|------------------|--------------|
| **Reciprocity** | Give first, unexpected | Free tools, extended trials, valuable content | "Download our free security audit template" |
| **Scarcity** | Limited availability | Beta access, limited spots, time-limited offers | "Only 50 early access spots remaining" |
| **Authority** | Expert credentials | Certifications, customer logos, case studies | "Trusted by 3 of the Fortune 10 banks" |
| **Consistency** | Small commitments | Progressive onboarding, micro-conversions | "Start with our 2-minute security assessment" |
| **Liking** | Similarity, compliments | Relatable language, shared values | "Built by security engineers, for security teams" |
| **Social Proof** | Others' behavior | User counts, testimonials, reviews | "Join 10,000+ companies protecting their customers" |
| **Unity** | Shared identity | Community, exclusive groups, mission | "Join the movement to end identity fraud" |

## Viral Loop Checklist

### Collaborative Loop (Calendly Model)
- [ ] Product requires multiple users to function
- [ ] Each use exposes non-users to the product
- [ ] Invitation process is frictionless
- [ ] Value is visible to invitee before signup
- [ ] Branding is visible on shared content

### Incentivized Loop (Dropbox Model)
- [ ] Reward is tied to core product value
- [ ] Dual-sided (both parties benefit)
- [ ] Unique referral tracking per user
- [ ] Progress dashboard visible to users
- [ ] Automated reward delivery
- [ ] Milestone bonuses for power referrers

### Content/UGC Loop (Loom Model)
- [ ] Users create shareable content
- [ ] Content showcases product value
- [ ] Sharing is one-click easy
- [ ] Branding is visible on shared content
- [ ] Viewers can easily become creators

## Landing Page Elements Checklist

### Above the Fold
- [ ] Clear, benefit-driven headline
- [ ] Supporting subheadline
- [ ] Primary CTA button
- [ ] Social proof (logos or count)
- [ ] Hero image or video
- [ ] No navigation distractions

### Body Content
- [ ] Problem agitation
- [ ] Solution presentation
- [ ] Feature-benefit breakdown
- [ ] Social proof (testimonials)
- [ ] Trust signals (security, compliance)
- [ ] FAQ section
- [ ] Final CTA

### Technical
- [ ] Mobile responsive
- [ ] Fast load time (<3 seconds)
- [ ] Clear value proposition
- [ ] Single focus (one action)
- [ ] Form fields minimized
- [ ] Thank you page with next steps

## Email Sequence Templates

### Onboarding Sequence (5 emails)

**Email 1 - Welcome (Immediate)**
```
Subject: Welcome to [Product] - Let's get you started

Hi [Name],

Welcome! Here's how to get value in the next 5 minutes:
[Quick start guide]

[CTA: Complete Setup]
```

**Email 2 - Tip (Day 2)**
```
Subject: Pro tip: [Specific feature]

Hi [Name],

Most successful users enable [feature] right away. Here's why:
[Benefit explanation]

[CTA: Try It Now]
```

**Email 3 - Social Proof (Day 4)**
```
Subject: How [Similar Company] achieved [result]

Hi [Name],

[Customer] was facing [similar challenge]. Here's how they solved it:
[Case study summary]

[CTA: Read Full Story]
```

**Email 4 - Value Reminder (Day 7)**
```
Subject: You're [X] away from [benefit]

Hi [Name],

You've made great progress! Complete [action] to unlock [benefit].

[Progress indicator]

[CTA: Complete Now]
```

**Email 5 - Upgrade/Feedback (Day 14)**
```
Subject: How has your experience been?

Hi [Name],

Quick question: How has [Product] worked for you so far?
[Reply to let us know]

P.S. Ready to upgrade? [Upgrade benefits]

[CTA: Upgrade Now]
```

## A/B Testing Ideas

### Headlines
- Problem-focused vs. Benefit-focused
- Specific numbers vs. General claims
- Question vs. Statement
- Short vs. Long

### CTAs
- Action verb vs. Benefit-focused
- First person vs. Second person
- Button color variations
- Urgency words (Now, Today, Free)

### Social Proof
- Logos vs. Testimonials
- Number of users vs. Specific results
- Video vs. Text testimonials
- Customer photos vs. No photos

### Forms
- Number of fields (3 vs. 5)
- Required vs. Optional fields
- Inline validation vs. Post-submit
- Progress indicators

## Crisis/Copy Fix Checklist

If conversion rates drop:

1. **Check Technical Issues**
   - [ ] Page loads correctly
   - [ ] Form submits properly
   - [ ] Mobile experience works
   - [ ] Tracking pixels firing

2. **Review Traffic Quality**
   - [ ] Source of traffic changed?
   - [ ] New campaign launched?
   - [ ] Audience targeting correct?

3. **Analyze Copy**
   - [ ] Headline still relevant?
   - [ ] CTA clear and compelling?
   - [ ] Social proof current?
   - [ ] Pricing displayed correctly?

4. **Check Competitors**
   - [ ] New competitor launched?
   - [ ] Competitor changed pricing?
   - [ ] Market conditions shifted?
